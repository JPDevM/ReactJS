# Subscription App

> Use this app for manage yours subscriptions payments.

### üöÄ Quick Start

In server folder, create a `.env` file and copy the content from `.env.example`.

#### Cosas para el servidor

Install node dependences.

```shell
$ cd server
$ npm i
```

Turn on the server.

```shell
$ cd src
$ npm start
```

Create database.

```shell
$ npm run dbStart
```

Install React dependences.

```shell
$ cd ../../client
$ npm i
```
